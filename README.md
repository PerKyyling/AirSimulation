**Симулятор потока воздуха**

В этом репозитории будут появляться файлы, для одного проекта - CFD солвера

**Структура**

*simulationOnPython.py* - 2d симуляция написанная только на python


*main.py* - последняя версия симуляции на данный момент. Сейчас из себя представляет гибридную программу: для расчетов - c++, для визуализации - python. Юзает готовые библиотеки из simulation.cpp
